## Installation

1. Clone the repository from your terminal:

   ```
   git clone https://github.com/ZeroWk-EM/ServerTS-Mongoose.git
   ```

2. Install the dependencies:

   ```
   npm install
   ```

3. Create `.env` file and set the required environment variables.

   ```
   PORT=
   MONGO_URL=
   SECRET_JWT=
   SALT_ROUND=
   ```

4. Start the server:
   ```
   npm  start
   ```
5. Test request:
   ```
   npm run test
   ```
